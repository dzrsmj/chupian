Privacy Policy for 出片 (Chupian)

Last updated: May 28, 2026

Data Collection

出片 does not collect, store, or transmit any personal information or user data.

All text processing and image generation happens locally on your device. The extension does not send any data to external servers.

Permissions

The extension requests the following permissions solely for its core functionality:

  • contextMenus — To add the "出片" option to the right-click menu
  • activeTab — To access the current tab's URL and title for source attribution on generated cards
  • scripting — To read selected text on the current page when the user invokes the extension

None of these permissions are used to collect, store, or share user data.

Third-Party Services

出片 does not use any third-party analytics, advertising, or tracking services.

Contact

If you have questions about this privacy policy, please open an issue at: github.com/dzrsmj/chupian
